# sheep
A front-end rendering solution , support IE6 , IE8 or modern browser

##### install

start server
```shell
cd server
npm install
node app
```

start client
```shell
cd resources\views
npm start
```

open http://127.0.0.1:8889/index in your browser 

theme from http://getbootstrap.com/examples/jumbotron-narrow/

##### Tools

Build By Visual Studio Code
